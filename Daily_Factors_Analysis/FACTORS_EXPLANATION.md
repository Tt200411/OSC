# ETT数据集每日因子说明文档

## 概述

本文档详细说明了对ETT-small数据集（ETTh1, ETTh2, ETTm1, ETTm2）的OT列（目标列）按日切分后计算的统计因子。

### 数据切分方式

- **ETTh系列** (ETTh1, ETTh2): 每天24个数据点（小时级数据）
- **ETTm系列** (ETTm1, ETTm2): 每天96个数据点（15分钟级数据）

---

## 因子详细说明

### 1. 基础统计量

#### 1.1 Mean (均值)
- **定义**: 每日OT值的算术平均值
- **计算公式**: `mean = Σ(x_i) / n`
- **物理意义**: 反映当天OT的平均水平
- **应用场景**:
  - 识别高负荷/低负荷日期
  - 长期趋势分析
  - 异常日检测

#### 1.2 Std (标准差)
- **定义**: 每日OT值的标准差
- **计算公式**: `std = sqrt(Σ(x_i - mean)² / n)`
- **物理意义**: 反映当天OT的离散程度
- **应用场景**:
  - 评估日内波动稳定性
  - 识别高波动日
  - 风险评估

#### 1.3 Median (中位数)
- **定义**: 每日OT值排序后的中间值
- **物理意义**: 不受极端值影响的中心位置
- **应用场景**:
  - 与均值对比识别偏态分布
  - 鲁棒的中心趋势估计

#### 1.4 Min / Max (最小值/最大值)
- **定义**: 每日OT的极值
- **物理意义**: 当天的负荷边界
- **应用场景**:
  - 容量规划
  - 峰谷分析

#### 1.5 Range (极差)
- **定义**: 最大值与最小值之差
- **计算公式**: `range = max - min`
- **物理意义**: 当天OT的变化幅度
- **应用场景**:
  - 评估日内变化剧烈程度
  - 调度复杂度评估

---

### 2. 变异性指标

#### 2.1 CV (变异系数)
- **定义**: 标准差与均值的比值
- **计算公式**: `cv = std / mean`
- **物理意义**: 归一化的波动性度量，消除量纲影响
- **应用场景**:
  - 跨数据集波动性对比
  - 相对稳定性评估
- **关键发现**:
  - ETTh2/ETTm2 (cv≈0.16): 相对稳定
  - ETTh1/ETTm1 (cv≈0.22-0.31): 波动较大

#### 2.2 IQR (四分位距)
- **定义**: 第75百分位数与第25百分位数之差
- **计算公式**: `iqr = Q3 - Q1`
- **物理意义**: 中间50%数据的分布范围
- **应用场景**:
  - 鲁棒的离散度度量
  - 异常值检测

---

### 3. 分布特征

#### 3.1 Skewness (偏度)
- **定义**: 分布的不对称性度量
- **物理意义**:
  - skewness > 0: 右偏（长尾在右侧）
  - skewness < 0: 左偏（长尾在左侧）
  - skewness ≈ 0: 对称分布
- **应用场景**:
  - 识别负荷分布模式
  - 预测模型选择

#### 3.2 Kurtosis (峰度)
- **定义**: 分布的尖峭程度
- **物理意义**:
  - kurtosis > 0: 尖峰分布（厚尾）
  - kurtosis < 0: 平坦分布（薄尾）
- **应用场景**:
  - 极端事件风险评估
  - 异常检测灵敏度调整

---

### 4. 趋势指标

#### 4.1 Trend (线性趋势斜率)
- **定义**: 一阶线性拟合的斜率
- **计算公式**: 通过最小二乘法拟合 `y = ax + b`，取斜率a
- **物理意义**:
  - trend > 0: 上升趋势
  - trend < 0: 下降趋势
  - trend ≈ 0: 平稳
- **应用场景**:
  - 日内趋势识别
  - 预测起始点选择
- **关键发现**:
  - ETTh2 (trend≈0.25): 明显上升趋势
  - ETTm1 (trend≈0.015): 基本平稳

#### 4.2 First-Last Diff (首尾差值)
- **定义**: 当天最后一个值减去第一个值
- **计算公式**: `diff = OT[-1] - OT[0]`
- **物理意义**: 当天的净变化量
- **应用场景**:
  - 快速趋势判断
  - 日间连续性分析

#### 4.3 First-Last Ratio (首尾比率)
- **定义**: 最后一个值与第一个值的比值
- **计算公式**: `ratio = OT[-1] / OT[0]`
- **物理意义**: 相对变化率
- **应用场景**:
  - 百分比变化分析
  - 增长率评估

---

### 5. 波动性指标

#### 5.1 Volatility (波动率)
- **定义**: 一阶差分的标准差
- **计算公式**: `volatility = std(diff(OT))`
- **物理意义**: 相邻时刻变化的剧烈程度
- **应用场景**:
  - 预测难度评估
  - 平滑参数选择
- **关键发现**:
  - ETTh系列 (0.82-1.16): 高波动
  - ETTm系列 (0.34-0.40): 低波动（更细粒度平滑了变化）

#### 5.2 Mean Abs Change (平均绝对变化)
- **定义**: 相邻时刻差值绝对值的均值
- **计算公式**: `mean_abs_change = mean(|diff(OT)|)`
- **物理意义**: 平均变化幅度
- **应用场景**:
  - 调度频率规划
  - 控制策略设计

#### 5.3 Max Abs Change (最大绝对变化)
- **定义**: 相邻时刻差值绝对值的最大值
- **计算公式**: `max_abs_change = max(|diff(OT)|)`
- **物理意义**: 最剧烈的单步变化
- **应用场景**:
  - 系统冲击评估
  - 安全裕度设计

---

### 6. 百分位数

#### P10, P25, P75, P90
- **定义**: 第10、25、75、90百分位数
- **物理意义**: 分布的关键分位点
- **应用场景**:
  - 容量规划（P90作为设计容量）
  - 服务水平协议（SLA）设定
  - 异常阈值确定

## 数据集对比分析

### ETTh1 vs ETTh2
| 特征 | ETTh1 | ETTh2 | 差异 |
|------|-------|-------|------|
| 平均负荷 | 13.33 | 26.59 | ETTh2约为ETTh1的2倍 |
| 标准差 | 1.48 | 3.55 | ETTh2波动更大 |
| 变异系数 | 0.22 | 0.16 | ETTh1相对波动更大 |
| 趋势 | 0.065 | 0.248 | ETTh2上升趋势更明显 |
| 波动率 | 0.82 | 1.16 | ETTh2变化更剧烈 |

### ETTm1 vs ETTm2
| 特征 | ETTm1 | ETTm2 | 差异 |
|------|-------|-------|------|
| 平均负荷 | 13.32 | 26.59 | 与h系列一致 |
| 标准差 | 1.48 | 3.54 | 与h系列一致 |
| 变异系数 | 0.31 | 0.16 | ETTm1相对波动更大 |
| 趋势 | 0.015 | 0.059 | 都较平稳 |
| 波动率 | 0.40 | 0.34 | 都较低（细粒度效应） |

### h系列 vs m系列
| 特征 | h系列 (24点/天) | m系列 (96点/天) | 差异 |
|------|----------------|----------------|------|
| 波动率 | 0.82-1.16 | 0.34-0.40 | m系列更平滑 |
| 趋势斜率 | 0.065-0.248 | 0.015-0.059 | m系列趋势更缓 |
| 数据粒度 | 粗 | 细 | m系列捕捉更多细节 |

**关键洞察**:
- 更细的时间粒度（m系列）导致相邻点变化更小，波动率降低
- h系列更适合捕捉宏观趋势
- m系列更适合精细化预测和控制

## 应用建议

### 1. 预测模型选择
- **高波动数据** (ETTh2, volatility=1.16): 使用鲁棒模型（如LSTM、Transformer）
- **低波动数据** (ETTm2, volatility=0.34): 简单模型即可（如ARIMA）

### 2. 特征工程
- **趋势明显** (ETTh2, trend=0.25): 添加趋势特征、时间特征
- **趋势平稳** (ETTm1, trend=0.015): 关注周期性特征

### 3. 异常检测阈值
- 使用 `mean ± 3*std` 作为初始阈值
- 结合IQR方法: `[Q1-1.5*IQR, Q3+1.5*IQR]`

### 4. 数据预处理
- **高CV数据**: 考虑归一化或标准化
- **偏态分布**: 考虑对数变换或Box-Cox变换

## 使用示例

### Python读取因子数据
```python
import pandas as pd

# 读取ETTh1每日因子
df = pd.read_csv('ETTh1_daily_factors.csv')

# 筛选高波动日
high_volatility_days = df[df['volatility'] > df['volatility'].quantile(0.9)]

# 筛选上升趋势日
uptrend_days = df[df['trend'] > 0]

# 计算月度平均因子
df['month'] = pd.to_datetime(df['date']).dt.to_period('M')
monthly_factors = df.groupby('month')[['mean', 'std', 'volatility']].mean()
```

### 因子用于预测
```python
# 将因子作为额外特征
features = df[['mean', 'std', 'cv', 'trend', 'volatility', 'range']]

# 预测下一天的因子
from sklearn.ensemble import RandomForestRegressor
model = RandomForestRegressor()
model.fit(features[:-1], df['mean'].shift(-1)[:-1])
```
